# 015-定义文章类型模板

## 背景
需要在平台之外，定义不同内容类型的写作模板，如感悟散文、工具文、办事攻略等。

## 目标
建立文章类型体系，定义各类文章的写作规范和提示词模板。

## 验收标准
- [ ] 定义文章类型列表及其特征
- [ ] 建立文章类型的数据结构
- [ ] 实现文章类型配置的加载和管理

## 实施记录

### 文章类型定义

#### 1. 感悟类散文
- 特征：情感表达为主，个人思考为辅
- 结构：场景引入+情感铺垫+思考升华
- 写作要点：真情实感，细节丰富，结尾升华

#### 2. 工具文/干货
- 特征：实用性强，步骤清晰，可操作
- 结构：痛点引入+方案介绍+步骤说明+效果展示
- 写作要点：简洁明了，步骤清晰，要有案例

#### 3. 办事攻略
- 特征：指导性明确，步骤详细，注意事项
- 结构：背景说明+前置条件+步骤流程+常见问题
- 写作要点：信息准确，步骤完整，提示到位

#### 4. 观点输出
- 特征：立场鲜明，论据充分，有启发性
- 背景引入+观点提出+论据支撑+升华总结
- 写作要点：观点新颖，论证严密，结尾有启示
