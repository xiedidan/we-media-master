# 自媒体文章写作Agent - 原始需求

## 1. 项目概述

开发一个AI Agent，辅助用户完成自媒体文章的写作，支持从素材收集到内容生成再到多平台发布的一站式服务。

## 2. 功能需求

### 2.1 内容输入
- **人工输入模式**: 用户提供主题、关键词或大纲，Agent负责扩写和润色
- **实时热点模式**: 接入搜索获取当前热点话题，作为写作素材

### 2.2 文章生成
- **长文支持**: 支持撰写1500-5000字的深度文章（公众号、知乎、今日头条等）
- **短内容支持**: 支持撰写200-500字的短内容（微博、小红书等）
- **多风格适配**: 根据不同平台自动调整文风（严肃/活泼/专业/通俗）
- **RSS订阅抓取**: 支持订阅RSS源抓取内容作为写作素材（v1.1+）

### 2.3 文章交付
- **纯文本输出**: Markdown格式输出，支持一键复制
- **多平台发布**: 支持一键发布到微信公众号、知乎、今日头条等平台（需要授权）

### 2.4 辅助功能
- **标题生成**: 根据内容自动生成多个吸引眼球的标题
- **摘要提取**: 自动提取文章核心观点生成摘要
- **配图建议**: 根据内容推荐合适的配图思路/关键词

## 3. 技术架构

### 3.1 Agent框架
- 采用 LangChain / LangGraph 构建Agent工作流
- 支持多模型接入（OpenAI / Claude / DeepSeek / 本地模型）

### 3.2 数据来源
- 搜索API获取实时热点
- 用户手动输入素材
- 浏览器爬虫获取指定网页内容

### 3.3 平台集成
- 微信公众号API
- 知乎API
- 今日头条号API
- 小红书
